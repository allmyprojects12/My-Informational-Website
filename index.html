المشروع الأول : وكالة الأوزري لتجارة إستيراد الدراجات النارية


وصف المشروع :


  • النوع:

    • نظام لإدارة المبيعات والمشتريات لشركة "الأوزري للتجارة والاستيراد" متخصص في الدراجات النارية وقطع الغيار (Shineray).

  • الوظيفة الأساسية:

    • يهدف النظام إلى إصدار فواتير البيع والشراء وتتبع الأصناف المباعة مثل الدراجات الكاملة (Shineray XY250GY) وقطع الغيار (محركات، إطارات).

    المشروع الثاني : نظام إدارة مكتب تأجير عقارات 


    وصف المشروع :


  • النوع:

    • نظام لإدارة الأعمال المالية والإدارية متكامل (MIS)، يهدف إلى تطوير النظام القائم لمعالجة القصور الهيكلي في الأتمتة المالية والأمن وتكامل البيانات الإدارية. 


  • الوظيفة الأساسية:

    • يركز النظام على أتمتة العمليات المالية المعقدة، وأهمها تطوير آلية لإنشاء جدول أقساط شهرية آلياً عند توقيع كل عقد إيجار. كما يهدف إلى تعزيز الأمان والمساءلة عبر إنشاء سجل تدقيق (Audit Log) لتتبع أنشطة المستخدمين، وتوفير وحدة حسابات وتقارير مالية جديدة لحساب صافي الأرباح والخسائر وأداء الوسطاء بشكل دوري. بالإضافة إلى ذلك، يعمل على تحسين إدارة المحتوى عبر إضافة إمكانية ربط وتخزين صور العقارات.
